# Contributing guide

Thank you for considering contributing to ProzillaOS! For bug reports and feature requests, open a [GitHub issue](https://github.com/prozilla-os/ProzillaOS/issues/new). If you have questions, reach out on [Discord](https://discord.gg/JwbyQP4tdz).

For setup instructions, code style, commit conventions, and everything else, see the [Development guide](https://os.prozilla.dev/docs/guides/development) ([source](./docs/src/guides/development.md)).
